## Getting Started

#### Step 1:
```bash
npm install
```
#### Step 2:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
